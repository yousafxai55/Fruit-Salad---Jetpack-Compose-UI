# 🍎 Fruit Salad - Jetpack Compose UI & Animations

Fruit Salad is a modern, high-performance Android application built entirely with **Jetpack Compose**. The app showcases a premium fruit delivery service with a focus on smooth UI/UX, advanced animations, and a clean Material 3 design.

## ✨ Features
- **Modern UI:** Built with Material 3 components and custom theming.
- **Advanced Animations:**
  - **3D Image Rotation:** Interactive fruit images that rotate in 3D space.
  - **Typewriter Effect:** Text that appears letter-by-letter for a premium feel.
  - **Explosion & Spin Overlays:** Custom transition effects when moving between screens.
  - **Water Ripple & Bobbing:** Dynamic animations on the welcome screen.
- **Edge-to-Edge Design:** Fully immersive experience with proper status bar handling.
- **Navigation:** Smooth transitions between Home, Fruit Detail, and Combo screens.

## 🛠 Tech Stack
- **Language:** Kotlin
- **UI Framework:** Jetpack Compose
- **Design System:** Material 3
- **Navigation:** Compose Navigation
- **Architecture:** MVVM (Model-View-ViewModel)

## 🚀 How to Run
1. Clone this repository:
2. Open the project in **Android Studio (Ladybug or newer)**.
3. Make sure you have **Kotlin 2.0+** configured.
4. Sync Gradle and run the app on an emulator or physical device.

## 📂 Project Structure
- `screen/`: Contains all UI screens (Home, Detail, Welcome).
- `animation/`: Custom animation logic and overlay components.
- `ui/theme/`: Material 3 color schemes, typography, and themes.
